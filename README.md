# Business OS — WhatsApp WebSocket Server

Persistent Node.js server for WhatsApp Web sessions via @whiskeysockets/baileys.

## Deploy to Railway
1. Push to GitHub
2. Connect to Railway
3. Set env vars: PORT, ALLOWED_ORIGIN
4. Deploy

## Environment variables
- PORT: WebSocket server port (default 3001)
- ALLOWED_ORIGIN: The main app URL (https://clickdz.cloud)
